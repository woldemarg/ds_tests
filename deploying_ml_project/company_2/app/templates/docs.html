Pipe API Documentation

by Volodymyr Holomb (as a test assignment)

The pipe API receives a JSON request containing dates within range from 2015-01-01 to 2020-01-01:

  • "from" - start of the period for prediction
  • "to" - end of the period for prediction

The pipe API returns a JSON response containing:

  • "jalan":"predictions" for each day within requested period
  • "rakuten":"predictions" for each day within requested period
